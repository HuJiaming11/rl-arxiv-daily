## 工作流程配置

### 1. 每日论文更新 (cv-arxiv-daily.yml)
- **触发方式**: 
  - 手动触发 (workflow_dispatch)
  - 定时触发: 每5天运行一次 (cron: "0 0 */5 * *")
- **Python版本**: 3.10
- **主要功能**: 
  - 从arXiv获取最新论文
  - 更新README.md和docs目录下的相关文件
  - 自动提交到GitHub仓库

### 2. 每周论文链接更新 (update_paper_links.yml)
- **触发方式**:
  - 手动触发 (workflow_dispatch)
  - 定时触发: 每周一上午8点 (cron: "0 8 * * 1")
- **主要功能**:
  - 更新论文的代码链接（注：PapersWithCode API已停用，此功能当前不可用）
  - 同样更新README.md和docs目录下的相关文件
  - 自动提交到GitHub仓库

## 论文关键词配置

### 第一阶段：算法关键词筛选
当前配置的论文搜索关键词包括：

1. **Reinforcement Learning**: Reinforcement, Reinforcement Learning, GRPO
2. **Imitation Learning & SFT**: Imitation Learning, Supervised Fine-tuning, Vision-Language-Action Fine-tuning, VLA Fine-tuning, Diffusion Policy, Flow-matching

### 第二阶段：领域二次筛选
在第一阶段获取的论文基础上，使用领域关键词进行二次筛选，只保留自动驾驶或机器人相关的论文：

**自动驾驶领域关键词**:
- Autonomous Driving, Self-driving, Autonomous vehicle, Autonomous vehicles
- Intelligent vehicle, Intelligent vehicles, Vehicle control
- Driving, Traffic, Lane, Road

**机器人领域关键词**:
- Robot, Robotics, Robotic, Manipulation, Navigation
- Mobile robot, Mobile robots, Robotic control, Robot control
- Embodied, Embodied AI

**筛选机制说明**:
1. **第一阶段**: 使用算法关键词从arXiv获取相关论文
2. **第二阶段**: 检查论文标题是否包含自动驾驶或机器人领域关键词
3. **结果**: 只保留同时满足算法和领域要求的论文，确保论文的相关性和针对性

**关键词说明**:
- 基础算法关键词：Reinforcement Learning, Imitation Learning等
- 智能驾驶领域关键词：Autonomous Driving, Self-driving, Vehicle Control, Intelligent Vehicles
- 机器人领域关键词：Robotics, Robot Navigation, Mobile Robot, Manipulation, Robotic Control
- **筛选机制**：通过OR逻辑连接所有关键词，确保论文同时满足算法和领域要求

## 后续步骤

### 1. GitHub Actions 启用
1. 进入仓库 Settings -> Actions -> Workflow permissions
2. 选择 "Read and write permissions" 并保存
3. 进入 Actions 页面，点击 "I understand my workflows, go ahead and enable them"
4. 在右侧边栏选择 "Run Arxiv Papars Daily" 并点击 "Enable workflow"
5. 点击 "Run workflow" 等待约1分钟直到任务完成
6. 对 "Run Update Paper Links Weekly" 重复相同操作

### 2. GitPage 设置 (可选)
1. 进入 Setting -> Pages -> Build an deployment
2. Source 选择 "Deploy from a branch"
3. Branch 选择 "main" 和 "/docs" 文件夹
4. 保存后可访问: https://hujiaming11.github.io/rl-arxiv-daily

### 3. 添加新关键词 (可选)
1. 编辑 `config.yaml` 中的 `keywords` 部分
2. 可以添加更多过滤器或关键词
3. 推送更改到远程仓库并手动重新运行GitHub Actions
